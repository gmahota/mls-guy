# Mls-Guy

**Mls-Guy** Mahota Living Style - Guy é um projeto web onde apresento o meu plano alimentar.

E posso consultar de forma rapida o que é permitido comer dentro da minha dieta alimentar.

É baseado na linguagem Markdown e React components ([MDX](/features/mdx)), gerados de forma automatica.
